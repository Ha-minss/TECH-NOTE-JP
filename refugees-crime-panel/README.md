# Refugee Inflows → Crime Rates

難民流入規模と犯罪率の変化との関係を、国・年パネルデータで分析した計量経済学プロジェクトです。

本プロジェクトでは、2013年から2022年までの30か国のデータを使用し、人口当たりの難民数が増加した国で犯罪率も同時に上昇するかを検討します。分析は単純な相関関係ではなく、国ごとの固定的特性と年ごとの共通ショックを統制するTwo-Way Fixed Effects構造で実施しました。

---

## 1. Overview

難民流入と犯罪率の関係は、社会的に非常に機微性の高いテーマです。単に「難民が増えた国では犯罪も増えた」という相関関係だけで結論を出すと、国ごとの制度差、景気変動、治安統計の基準、人口規模、長期トレンドをすべて混同する可能性があります。

本プロジェクトではこの問題を避けるため、国・年パネルデータを構築し、難民流入と複数の犯罪類型との関係を検討しました。

分析対象は2013年から2022年までの30か国で、最終データは300件の国・年観測値から構成されます。主な説明変数は人口10万人当たりの難民数であり、結果変数は人口10万人当たりの犯罪率です。

主な結果は次のとおりです。

基本的なTwo-Way Fixed Effects分析では、財産犯罪率に正の相関関係が見られました。一方、暴力犯罪、薬物犯罪、性犯罪では、一貫した正の関係を確認することは困難でした。

また、国別の長期トレンドを追加すると、財産犯罪における正の関係も大きく弱まりました。したがって、本プロジェクトの結論は「難民流入が犯罪を増加させる」というものではなく、**基本仕様では一部に正の相関が観察されるものの、頑健な因果効果として解釈することは難しい**というものです。

---

## 2. Problem & Objective

本プロジェクトの出発点は次の問いです。

> 難民流入規模が拡大した国では、犯罪率も同時に上昇するのか？

この問いは単純に見えますが、実際には非常に慎重に扱う必要があります。難民流入は無作為に発生しません。特定国の経済状況、政治条件、国境の位置、既存移民ネットワーク、制度的受入能力、社会福祉水準によって難民流入規模は異なる可能性があります。

犯罪率についても、国ごとに測定方法や届出基準が異なる可能性があります。犯罪の届出率が高い国もあれば、警察統計の集計方法が異なる国もあります。また特定年には、景気後退、新型コロナウイルス、制度変更など、複数国へ同時に影響する出来事が発生する可能性があります。

したがって、単純な国間比較だけでは十分ではありません。

本プロジェクトの目的は次のとおりです。

第一に、難民流入と犯罪率を国・年パネル構造に整理します。

第二に、国ごとの固定的特性と年ごとの共通ショックを統制した後にも、難民流入と犯罪率の関係が残るかを確認します。

第三に、犯罪を一つにまとめず、暴力犯罪、財産犯罪、薬物犯罪、性犯罪に分け、結果が特定の犯罪類型だけに現れるかを確認します。

第四に、lag、lead placebo、国別トレンド、特定年の除外、first differenceを通じて、基本結果の安定性を検討します。

本プロジェクトは特定の政治的主張を裏付けるための分析ではなく、**機微性の高い社会的主張をデータで検討する際に、どのような検証が必要かを示すプロジェクト**です。

## 3. Data

分析には、2013年から2022年までの国・年パネルデータを使用しました。

分析単位は、一つの国が特定年に持つ難民流入規模と犯罪率です。

- 分析期間：2013～2022年
- 分析単位：国 × 年
- 国数：30か国
- 最終観測値数：300件
- 主な説明変数：人口10万人当たりの難民数
- 主な結果変数：人口10万人当たりの犯罪率

元データには以下の情報が含まれています。

- 国名と年
- 移民数
- 難民数
- 人口
- GDP関連指標
- 社会保護支出関連指標
- 失業率
- 高等教育参加規模
- 暴力犯罪件数
- 財産犯罪件数
- 薬物犯罪件数
- 性犯罪件数
- 警察人員数

分析では国の規模差を小さくするため、難民数と犯罪件数を人口10万人当たりの比率へ変換しました。国ごとに人口規模が大きく異なるため、元データの絶対人数をそのまま比較すると、大国ほど難民数と犯罪件数の双方が大きく見える問題が生じます。

また、犯罪率と難民流入変数には対数変換を使用しました。これは国間の規模差を小さくし、係数をおおよその比率変化として解釈しやすくするためです。

結果変数は以下の4つで構成しました。

- 暴力犯罪率
- 財産犯罪率
- 薬物犯罪率
- 性犯罪率

統制変数として、GDP関連指標、社会保護支出、失業率、高等教育参加率、警察人員比率を使用しました。これらの変数は、国の経済・社会条件と治安能力が犯罪率へ影響する可能性を考慮するためのものです。

## 4. Method / System Design

中心的な分析方法はTwo-Way Fixed Effectsです。

本プロジェクトでは、国Fixed Effectsと年Fixed Effectsを併せて使用しました。

国Fixed Effectsは、各国が元来持つ固定的な差を統制します。例えば犯罪統計の集計方法、法制度、地理的位置、長期的な社会構造など、分析期間中に大きく変化しない差が該当します。

年Fixed Effectsは、特定年に複数国へ共通して影響するショックを統制します。例えば新型コロナウイルス流行期、欧州全体の景気変動、国際情勢の変化、犯罪届出環境の変化などが含まれます。

基本モデルは次の関係を検討します。

> 同じ国の中で、特定年に人口当たりの難民数が増加したとき、犯罪率も同時に上昇するか？

さらに、以下の検証を実施しました。

第一に、現在の難民数だけでなく、1年lagと2年lagを使用しました。難民流入の影響が直ちに現れず、時間差を伴って現れる可能性を確認するためです。

第二に、1年leadをplaceboとして使用しました。将来の難民数が現在の犯罪率を説明する場合、難民流入が犯罪を増加させたという解釈よりも、共通トレンドや逆因果の可能性を疑うべきシグナルになります。

第三に、国別の線形トレンドを追加しました。国ごとに犯罪率が長期的に上昇・低下する傾向が異なる可能性があるためです。

第四に、特定年を除外するrobustness検証を実施しました。特定の出来事や期間が結果を主導しているかを確認するためです。

第五に、first difference分析を実施しました。国ごとの水準差を除去し、前年比の変化量同士の関係を確認するためです。

## 5. Implementation

本プロジェクトでは再現可能な分析のため、データ前処理、基本分析、動学的検証、robustness検証、可視化をそれぞれ別のスクリプトに分離しました。

全体の流れは次のとおりです。

1. 元データを読み込み、人口10万人当たりの比率と対数変数を生成します。
2. 基本Two-Way Fixed Effects分析を実行します。
3. lag、lead placebo、joint dynamic modelを実行します。
4. 国別トレンドを追加したrobustness検証を実施します。
5. 特定年除外検証とfirst difference分析を実行します。
6. 主な係数を可視化します。

主な実行ファイルは次のとおりです。

- `00_prepare_panel.py`  
  元データを読み込み、人口10万人当たりの比率と対数変数を生成します。

- `01_main_twfe_refugees_only.py`  
  基本Two-Way Fixed Effects分析を実行します。

- `02_dynamic_lags_leads_placebo.py`  
  現在値、1年lag、2年lag、1年lead placebo、joint dynamic modelを実行します。

- `03_robustness_country_trends.py`  
  国別線形トレンドを追加したrobustness検証を実行します。

- `04_robustness_drop_years.py`  
  特定年を除外した場合にも結果が維持されるかを確認します。

- `05_first_difference.py`  
  前年比変化量を使用したfirst difference分析を実行します。

- `06_make_coefplot.py`  
  主な結果を係数グラフとして可視化します。

- `run_all.py`  
  分析pipeline全体を一括実行し、結果表を出力します。

共通で使用する変数生成関数と回帰実行関数は、`src/ref_crime/`配下へ分離しました。この構造は、分析結果を一度作成して終わるのではなく、同一コマンドで再生成できるようにするためのものです。

## 6. Evaluation

基本Two-Way Fixed Effectsの結果は、結果変数によって異なりました。

財産犯罪率では約0.058の正の係数が見られ、統計的に有意でした。

一方、暴力犯罪率、薬物犯罪率、性犯罪率では、統計的に明確な正の関係を確認することは困難でした。

基本結果は次のように要約できます。

| 結果変数 | 基本TWFE結果 | 解釈 |
|---|---:|---|
| 暴力犯罪率 | 約+0.226 | 統計的に明確ではない |
| 財産犯罪率 | 約+0.058 | 正の相関関係を観察 |
| 薬物犯罪率 | 約+0.014 | 統計的に明確ではない |
| 性犯罪率 | 約+0.067 | 統計的に明確ではない |

しかし、この結果だけで結論を出すことはしませんでした。

動学的検証では、財産犯罪率について現在値とlag値だけでなく、lead placeboにも正の関係が見られました。これは重要な警告シグナルです。将来の難民数が現在の犯罪率とも関連して見える場合、基本結果は因果効果ではなく、国別トレンドや共通要因を反映している可能性があるためです。

国別線形トレンドを追加すると、基本仕様で有意だった財産犯罪率の正の関係は弱まりました。財産犯罪率の係数は約0.010へ低下し、統計的に有意ではなくなりました。

暴力犯罪と薬物犯罪については、複数の仕様で一貫した正の関係を確認することが困難でした。性犯罪では、一部のrobustness仕様で負の係数が見られましたが、これも強い結論として解釈するより、仕様感応性を示す結果とみなす方が適切です。

したがって、全体的な評価は次のとおりです。

> 基本TWFEでは財産犯罪率に正の相関関係が観察されるが、lead placeboと国別トレンドを考慮すると、難民流入の頑健な因果効果として解釈することは難しい。  
> 暴力犯罪、薬物犯罪、性犯罪では、一貫した正の関係を確認することは困難である。

## 7. Key Design Decisions

### なぜ人口10万人当たりの比率を使用したのか？

国ごとに人口規模が大きく異なるため、難民数や犯罪件数を元データのまま比較すると、大国ほど値が大きくなります。

本プロジェクトでは国間の規模差を小さくするため、難民数と犯罪件数を人口10万人当たりの比率へ変換しました。

### なぜ対数変換を使用したのか？

難民流入規模と犯罪率は、国ごとの分布差が大きくなっています。一部の国では値が非常に大きく、別の国では非常に小さい場合があります。

対数変換を使用することで、極端値の影響を小さくし、結果をおおよその比率変化として解釈しやすくなります。

### なぜTwo-Way Fixed Effectsを使用したのか？

単純なregressionでは、国ごとの制度差や年ごとの共通ショックを十分に統制できません。

本プロジェクトでは国Fixed Effectsと年Fixed Effectsを併用し、国ごとの固定的特性と特定年の共通ショックを統制しました。

### なぜlead placeboを含めたのか？

難民流入が犯罪率を増加させるという解釈が正しければ、将来の難民数が現在の犯罪率を説明してはなりません。

lead placeboに関係が見られる場合、結果は因果効果ではなく、事前トレンド、共通ショック、逆因果の可能性を反映しているかもしれません。

本プロジェクトでは、財産犯罪におけるlead placeboが重要な警告シグナルとなったため、基本TWFE結果を強い因果効果として解釈しませんでした。

### なぜ国別トレンドを追加したのか？

国ごとに犯罪率が長期的に上昇・低下する方向は異なる可能性があります。この国別トレンドを統制しなければ、難民流入と犯罪率が同じ方向へ動いているように見える場合があります。

国別線形トレンドを追加した結果、基本仕様で有意だった財産犯罪の正の関係が大きく弱まりました。この点は結果解釈において非常に重要です。

## 8. Development Notes

本プロジェクトは当初、難民流入と犯罪率の基本的な相関関係を確認することから始まりました。

基本TWFE結果だけを見ると、財産犯罪率に正の関係が見られました。しかし、機微性の高い社会的テーマにおいて、基本回帰結果一つだけで結論を出すのは危険だと判断しました。

そこで分析の焦点は次第に、「効果があるか」を問う方向から、「この関係はどの程度安定しているか」を確認する方向へ変わりました。

第一の転換点はlead placeboでした。財産犯罪について、将来の難民流入変数も現在の犯罪率と関連して見えたため、基本結果を因果効果として解釈することが難しいというシグナルが現れました。

第二の転換点は国別トレンドでした。国別の長期トレンドを追加すると、財産犯罪の正の関係が弱まりました。これは基本結果が国別の長期的な動きの影響を受けていた可能性を示します。

第三の転換点は、結果の解釈方法でした。当初は「どの犯罪類型に正の関係があるか」を中心に見ることもできましたが、最終的には「頑健な正の関係があるか」を中心とする結論へ修正しました。

この過程を通じて、READMEと分析結果の中心メッセージを次のように整理しました。

> 基本仕様では一部に正の相関が見えるものの、複数の検証を経ると、難民流入が犯罪を増加させるという強い結論を支持することは難しい。

## 9. Limitations

本プロジェクトは国・年パネルデータを使用した観察研究です。そのため、結果を強い因果効果として解釈するには限界があります。

第一に、難民流入は無作為に発生しません。特定国の制度、経済状況、国境条件、既存移民ネットワークによって難民流入規模が異なる可能性があります。

第二に、国別の犯罪統計は集計方法と届出率が異なる可能性があります。同じ犯罪類型でも、国ごとに法的定義や届出慣行が異なる場合があります。

第三に、分析単位が国・年であるため、地域単位の詳細な変化を確認できません。難民が特定都市・地域に集中したか、犯罪率の変化が同じ地域で発生したかは分かりません。

第四に、標本は30か国・10年間で構成されており、より長い期間またはより広い国標本を使用した検証が必要です。

第五に、本分析は難民政策のすべての効果を評価するものではなく、犯罪率との関係だけを限定的に検討します。

第六に、基本TWFE構造には観測可能な統制変数と固定効果が含まれますが、時間とともに変化するすべての国別要因を完全に統制することはできません。

したがって、本プロジェクトの結論は次のように限定的に解釈する必要があります。

> 本データと仕様では、難民流入が暴力犯罪、薬物犯罪、性犯罪を一貫して増加させるという証拠を確認することは困難である。  
> 財産犯罪では基本仕様に正の関係が見られるが、lead placeboと国別トレンド検証を考慮すると、強い因果効果として解釈することは難しい。


## 12. What This Project Demonstrates

本プロジェクトは、機微性の高い社会的主張をデータで検討する際に必要な分析姿勢を示します。

第一に、単純な国間比較ではなく、国・年パネル構造として問題を再定義しました。

第二に、難民数と犯罪件数を人口10万人当たりの比率へ変換し、国の規模差を小さくしました。

第三に、国Fixed Effectsと年Fixed Effectsを使用し、国ごとの固定的特性と年ごとの共通ショックを統制しました。

第四に、暴力犯罪、財産犯罪、薬物犯罪、性犯罪を分け、結果が特定の犯罪類型だけに現れるかを確認しました。

第五に、lag、lead placebo、国別トレンド、特定年の除外、first differenceを通じて、基本結果の安定性を検討しました。

第六に、結果を誇張せず、限定的に解釈しました。基本仕様で一部に正の関係が見られても、検証過程で弱まる結果を強い因果効果として主張しませんでした。

本プロジェクトの中心は、単に回帰分析を実行したことではなく、**機微性の高い社会データにおいて結果をどのように検証し、慎重に解釈すべきかを示したこと**です。
