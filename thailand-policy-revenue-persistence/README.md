# 消費支援政策は、売上指標をどの程度長く動かすのか？

タイのHalf-Half系列の消費支援政策後に、**宿泊・飲食サービス業の指標が低曝露業種と比べてどのように反応し、その反応が何か月残ったのか**を分析した政策評価プロジェクトです。

本プロジェクトでは、政策により増加した売上を、そのまま持続的な返済能力として解釈しません。政策実施当月の反応、1～3か月のlag反応、累積反応、synthetic comparator、placebo、robustnessを併せて確認し、**政策による売上と持続需要のシグナルを区別する分析フロー**を設計しました。

---

## 1. Overview

政府の消費クーポンは、小規模事業者の売上を短期間で押し上げる可能性があります。しかし、政策によって増えた売上が政策終了後にも残るかは別の問題です。

政策期間中にのみ発生した注文であれば一時的需要である可能性があり、終了後にも維持されるのであれば、新規顧客の流入や反復需要のシグナルになり得ます。

本プロジェクトでは、タイのHalf-Half系列の消費支援政策後に、宿泊・飲食サービス業の指標がどのように反応したかを分析します。宿泊・飲食サービス業を政策曝露度の高い業種とし、金融業・不動産業・専門サービス業を低曝露の比較業種として設定しました。

中心的な問いは次のとおりです。

> 消費支援政策後、宿泊・飲食サービス業の指標はどの程度強く反応し、  
> その反応は政策終了後にも維持されたのか？

分析の結果、政策実施当月の反応は明確ではなかった一方、1～3か月lagの係数はすべて正の方向でした。0～3か月の累積反応は0.1667で、対数係数を割合へ換算すると約+18.1％の正の相対反応に相当します。

ただし、この結果をHalf-Half政策の純粋な因果効果として解釈してはいません。公開データには加盟店別売上やクーポン使用有無が含まれず、全国規模の政策であるため完全にクリーンな対照群も存在しないからです。

したがって、本プロジェクトの結論は次のとおりです。

> 消費支援政策の時期に、宿泊・飲食サービス業の指標は低曝露業種と比べて1～3か月にわたる正の相対反応を示した。  
> ただし、この反応には政策効果、観光需要、季節性、業種回復の流れが混在している可能性があるため、プラットフォーム融資では最近の売上増加をそのまま持続的な返済能力として解釈してはならない。

---

## 2. Problem & Objective

小売・飲食店・小規模事業者向け融資では、直近売上は返済能力を判断する中心指標のように見えます。直近売上が増加すれば、事業が回復し、融資を返済する能力も高まったと解釈しやすいためです。

しかし、政策による売上は別に考える必要があります。政府の消費クーポン、プラットフォームのプロモーション、観光需要、年末消費、景気回復が同時に作用すると、売上が実際の営業力以上に膨らむ可能性があります。

このとき、単純な売上増加率だけを見ると、2つの判断ミスが生じます。

| 誤った判断 | 生じる問題 |
|---|---|
| 政策期間中の売上をすべて持続売上と解釈 | 政策終了後に急速に消える可能性のある需要を過大評価 |
| 政策による売上をすべて一時効果と解釈 | 実際に顧客基盤が拡大した業種や店舗を過小評価 |

したがって重要な問いは、「売上が増えたか」ではなく、**増加した売上がどの程度長く残ったか**です。

本プロジェクトの目的は次のとおりです。

第一に、宿泊・飲食サービス業指標と低曝露比較業種指標の相対差を構成します。

第二に、消費支援政策の強度を月別に定義し、政策実施当月と1～3か月後の反応を分離します。

第三に、単純な比較業種平均とsynthetic comparatorを併用し、結果が一つの比較基準だけに依存していないかを確認します。

第四に、placeboとrobustness検証を通じて、結果をどこまで解釈できるかを確認します。

第五に、この結果を、プラットフォーム融資において政策による売上をどのように解釈すべきかへ接続します。

---

## 3. Data

公開データだけでは、加盟店別売上、クーポン使用有無、反復注文、返済結果を直接観測できません。

そのため本分析では、個別加盟店の信用リスクを推定するのではなく、公開月次業種データから確認可能な**宿泊・飲食サービス業の相対反応**を分析するよう設計しました。

分析対象は次のとおりです。

| 項目 | 内容 |
|---|---|
| 分析単位 | 月次業種指標 |
| 高曝露業種 | 宿泊・飲食サービス業 |
| 低曝露比較業種 | 金融業、不動産業、専門サービス業 |
| 中心的な比較値 | 宿泊・飲食サービス業と低曝露業種間の対数差 |
| 政策変数 | 月別政策実施日数／当該月の総日数 |
| 統制変数 | CPI、外国人観光客数、宿泊稼働率、非居住者支出 |
| 季節性統制 | 月別季節性Fixed Effects |

中心的なoutcomeは、宿泊・飲食サービス業と低曝露業種の相対差です。

```text
Gap_t = log(AFS_t) - log(LowExposure_t)
```

ここで`AFS`は宿泊・飲食サービス業指標、`LowExposure`は政策曝露が相対的に低いとみなした比較業種指標です。

このgapが大きくなることは、宿泊・飲食サービス業が比較業種より相対的に強く動いたことを意味します。つまり、単に「宿泊・飲食サービス業の指標が上昇した」ことではなく、他業種の動きを考慮してもより強い反応があったかを見る構造です。

比較業種は2つの方法で構成しました。

| 比較基準 | 構成方法 | 目的 |
|---|---|---|
| Low-exposure average | 金融業・不動産業・専門サービス業の単純平均 | 直感的な基準比較 |
| Synthetic comparator | 政策前期間の動きをより近づけた合成基準 | 結果が特定の比較群構成だけに依存しないかを確認 |

## 4. Method / System Design

本プロジェクトの中心は、政策の有無を単純な0/1ダミーとして扱わないことです。

政策は必ずしも1か月全体にわたり実施されるとは限らず、月末に開始された政策は当月平均指標へ直ちに大きく反映されない可能性があります。そのため、政策変数を月別の政策強度として定義しました。

```text
PolicyIntensity_t = 当該月の政策実施日数 / 当該月の総日数
```

その後、政策強度のlagを含む次の構造を使用しました。

```text
Gap_t = α + β0 PolicyIntensity_t
          + β1 PolicyIntensity_{t-1}
          + β2 PolicyIntensity_{t-2}
          + β3 PolicyIntensity_{t-3}
          + θ Controls_t
          + Month FE
          + ε_t
```

各係数の意味は次のとおりです。

| 係数 | 意味 | 解釈 |
|---|---|---|
| β0 | 政策実施当月の反応 | 政策開始月に直ちに現れる反応 |
| β1 | 1か月後の反応 | 政策から1か月後に残る反応 |
| β2 | 2か月後の反応 | 政策から2か月後に残る反応 |
| β3 | 3か月後の反応 | 政策から3か月後に残る反応 |
| β0+β1+β2+β3 | 0～3か月の累積反応 | 政策による反応が何か月累積するかを確認 |

本プロジェクトでは、単一の当月係数より0～3か月の累積反応を重視しました。

```text
Cumulative Response 0:3 = β0 + β1 + β2 + β3
```

この値は、政策による反応が当月だけにとどまったのか、それとも数か月間累積的に残ったのかを判断する基準です。

ただし、このモデルは純粋な政策効果を完全に識別する構造ではありません。結果は、「Half-Half政策が売上を正確にどの程度増加させたか」ではなく、**政策時期に宿泊・飲食サービス業指標が低曝露業種と比べてどのような動学的相対反応を示したか**として解釈しました。

## 5. Implementation

本プロジェクトは、政策による売上反応を一回の回帰結果だけで示さず、比較基準とrobustnessを併せて確認する構造に整理しました。

実装全体の流れは次のとおりです。

1. 月次業種指標を読み込みます。
2. 宿泊・飲食サービス業と低曝露比較業種の指標を構成します。
3. `Gap_t = log(AFS_t) - log(LowExposure_t)`を計算します。
4. 政策実施日数を月別政策強度へ変換します。
5. 政策強度の0～3か月lag変数を生成します。
6. CPI、観光客数、宿泊稼働率、非居住者支出などの統制変数を結合します。
7. lag modelを推定し、当月反応と累積反応を分離します。
8. synthetic comparatorを基準として同じ分析を繰り返します。
9. placeboとrobustness検証を実施します。
10. 結果をプラットフォーム融資における売上解釈基準へ接続します。

この実装構造の目的は、単一回帰結果を示すことではなく、政策による売上反応を次の順序で再現可能に確認することです。

```text
相対gapの構成
→ policy lag model
→ 0～3か月の累積反応
→ synthetic comparatorとの比較
→ placebo/robustness
→ プラットフォーム融資への解釈
```

---

## 6. Evaluation

評価は、「政策が売上を増加させたか」という一つの問いで終わらせませんでした。

本プロジェクトでは、次の順序で結果を解釈しました。

1. 宿泊・飲食サービス業指標は、低曝露業種と比べてどのように動いたか？
2. 政策実施当月の反応と1～3か月lagの反応はどのように異なるか？
3. 累積反応は正の方向か？
4. synthetic comparatorでも同じ方向が維持されるか？
5. placeboとrobustnessを考慮した場合、どこまで解釈できるか？
6. この結果をプラットフォーム融資へどのように活用すべきか？

---

### 6.1 政策時期には単純な売上増加率より相対反応を見る必要がある

消費支援政策後に宿泊・飲食サービス業指標が上昇しても、直ちに政策効果とはみなせません。同じ時期に観光需要、季節性、景気回復、他業種の動きが同時に変化する可能性があるためです。

そこで本プロジェクトでは、宿泊・飲食サービス業単独の指標ではなく、低曝露業種に対する相対差を使用しました。

| 比較方法 | 見る値 | 解釈 |
|---|---|---|
| 単純な業種指標 | 宿泊・飲食サービス業指標そのもの | 業種が上昇したかは分かるが、共通の景気動向と区別しにくい |
| 相対gap | 宿泊・飲食サービス業 - 低曝露業種 | 他業種の動きを考慮しても、より強く動いたかを確認可能 |

gapが大きくなることは、宿泊・飲食サービス業が金融業・不動産業・専門サービス業平均と比べて相対的に強く動いたことを意味します。そのため、このgapを基準に政策時期の相対反応を分析しました。

---

### 6.2 政策実施当月の反応は明確ではなく、lag反応がより重要だった

政策強度の0～3か月lagを含む結果は次のとおりです。

| 項目 | Estimate | 割合での解釈 | SE | p-value | 解釈 |
|---|---:|---:|---:|---:|---|
| 政策実施当月 β0 | -0.0036 | 約-0.4% | 0.0325 | 0.913 | 即時反応は明確ではない |
| 1か月後 β1 | 0.0440 | 約+4.5% | 0.0227 | 0.053 | 正の反応 |
| 2か月後 β2 | 0.0584 | 約+6.0% | 0.0325 | 0.072 | 正の反応 |
| 3か月後 β3 | 0.0679 | 約+7.0% | 0.0316 | 0.032 | 正の反応 |
| 0～3か月累積反応 | 0.1667 | 約+18.1% | 0.0602 | - | 累積反応は正の方向 |

政策実施当月の係数はほぼ0に近く、有意ではありませんでした。つまり、政策が開始された月に、宿泊・飲食サービス業が低曝露業種より直ちに大きく上昇したとは言いにくい結果です。

一方、1～3か月lagの係数はすべて正の方向でした。特に3か月後の係数は有意であり、0～3か月の累積反応も正の方向でした。

このパターンは政策の作動方式とも整合します。消費支援政策が発表されても、消費者が政策を認知し、実際に利用し、その利用がプラットフォーム注文と業種指標へ反映されるまでには時間がかかる可能性があります。

特に政策が月末に始まった場合、当該月全体を基準にすると即時反応は小さく見える可能性があります。そのため、政策開始月の売上だけを見ると反応を過小評価する可能性があります。

---

### 6.3 0～3か月の累積反応は正の方向だった

政策による売上を融資判断へ使用するには、当月反応だけでは不十分です。政策終了後にも指標が数か月維持されるかを確認する必要があります。

本プロジェクトでは、β0～β3を合計した0～3か月の累積反応を中心指標としました。

| 累積指標 | 値 | 割合での解釈 | 判断 |
|---|---:|---:|---|
| β0 + β1 + β2 + β3 | 0.1667 | 約+18.1% | 0～3か月の累積相対反応は正の方向 |

0～3か月の累積反応は0.1667で、対数係数を割合へ換算すると約+18.1％の正の相対反応に相当します。

これは、宿泊・飲食サービス業が政策時期に低曝露業種と比べて、数か月にわたる正の反応を示したことを意味します。

ただし、重要なのは「政策が売上を18.1％増加させた」という意味ではないことです。全国規模の政策であり、公開データのみを使用したため、純粋な政策効果を完全に分離できません。

より正確な解釈は次のとおりです。

> 政策時期に、宿泊・飲食サービス業指標が低曝露業種と比べて0～3か月にわたる正の相対反応を示した。

---

### 6.4 Synthetic comparatorでも方向は維持された

前述の結果が特定の比較基準だけに依存していないかを確認するため、低曝露業種の単純平均だけでなくsynthetic comparatorも併用しました。

synthetic comparatorは、政策前期間の動きをより近づけた比較基準です。

| 比較基準 | 0～3か月累積反応 | 解釈 |
|---|---:|---|
| 低曝露業種平均 | 正の方向 | 基本比較基準でも累積反応を確認 |
| Synthetic comparator | 正の方向 | 比較基準を変更しても主要パターンを維持 |

両基準において、0～3か月の累積反応は正の方向を維持しました。

つまり、「政策実施当月よりlagの累積反応が重要」という主要パターンは、比較基準を変更しても大きく変わりませんでした。

ただし、synthetic comparatorも完全な実験対照群ではありません。結果が特定の比較業種構成だけに依存しないかを確認するrobustness手段として解釈するのが適切です。

---

### 6.5 Placeboは過剰解釈を防ぐ警告シグナルだった

placebo testでは、政策が存在しなかった偽の時点を設定し、類似したgap変化が偶然にも現れ得るかを確認しました。

placebo結果では、一部の偽時点でもgap変化が現れました。これは主要結果を完全に否定するというより、結果を純粋な政策効果として過剰解釈すべきではないという警告とみなすのが適切です。

したがって、本分析では次のような結論を避けました。

> Half-Half政策が宿泊・飲食サービス業の売上を正確にX％増加させた。

代わりに、次のように限定して解釈しました。

> 政策時期に、宿泊・飲食サービス業指標が低曝露業種と比べて数か月にわたる正の相対反応を示したが、この反応には政策効果だけでなく、観光・季節性・業種回復の流れが混在している可能性がある。

---

### 6.6 Robustness検証でも累積反応の中央値は正の方向を維持した

さらに、比較群構成、lag長、outcome変換、統制変数の組合せ、washout期間、policy block定義を変更したrobustness検証を実施しました。

robustness検証の結果、累積policy-lag反応の中央値は正の方向を維持しました。

| 検証項目 | 目的 | 解釈 |
|---|---|---|
| 比較群構成の変更 | 低曝露業種の選択により結果が変わるかを確認 | 主要方向は概ね維持 |
| lag長の変更 | 反応持続期間の設定に敏感かを確認 | 累積反応の確認に重要 |
| outcome変換の変更 | 対数gapの定義だけに依存するかを確認 | 結果の解釈範囲を確認 |
| 統制変数の組合せ変更 | CPI・観光・稼働率の統制に敏感かを確認 | 統制方法による安定性を確認 |
| washout期間の変更 | 政策直後の移行期処理に敏感かを確認 | 動学的反応の解釈を補助 |
| policy block定義の変更 | 政策期間の定義に敏感かを確認 | 政策強度構成の感応度を確認 |

したがって、本結果は「政策が売上を正確にどの程度増加させたか」ではなく、**消費支援政策の時期に、宿泊・飲食サービス業指標が低曝露業種と比べて数か月にわたる正の相対反応を示し、その解釈には政策・観光・季節性を併せて考慮する必要がある**という結論として読むのが適切です。

---

### 6.7 融資では直近売上増加をそのまま返済能力として解釈してはならない

本分析は個別加盟店の返済リスクを直接予測したものではありません。しかし、プラットフォーム融資で売上データを解釈する際の重要な基準を示します。

政策時期には関連業種の指標が当月よりも1～3か月にわたり累積的に反応する可能性があります。この反応には政策効果だけでなく、観光需要、年末消費、業種回復の流れが同時に混在する可能性があります。

融資において、直近売上の増加をそのまま返済能力へ反映するのは危険です。

| 売上パターン | 融資上の解釈 | 運用判断 |
|---|---|---|
| 政策期間中のみ急増し、その後急速に低下 | 一時的な政策性売上の可能性 | 限度額算定で割引、またはモニタリング |
| 政策終了後にも一部維持 | 反復需要または顧客基盤拡大の可能性 | 限定的に返済能力へ反映 |
| 非割引注文と反復顧客が同時に増加 | 持続需要の可能性が高い | 追加信用評価feature候補 |
| 政策性売上比率が高く、維持率が低い | 返済能力を過大評価するリスク | 保守的な限度額または短期満期 |

つまり、重要なのは売上が増加したかではなく、**その売上がどの程度長く残るか**です。

## 7. Key Design Decisions

### なぜ宿泊・飲食サービス業単独の指標ではなく相対gapを使用したのか？

政策時期に宿泊・飲食サービス業指標が上昇しても、それが政策によるものか、観光回復や季節性によるものかを区別するのは困難です。

そこで宿泊・飲食サービス業と低曝露業種の対数差を使用しました。この方法は、他業種の共通動向を考慮した後にも、宿泊・飲食サービス業が相対的に強く動いたかを確認するための選択です。

### なぜ政策変数を0/1ダミーではなく政策強度として定義したのか？

政策は常に1か月全体にわたり実施されるとは限りません。月末に開始された政策は、当該月全体の指標では反応が小さく見える可能性があります。

そこで当該月の政策実施日数を月の総日数で割った政策強度を使用しました。

### なぜlag modelを使用したのか？

政策反応は実施当月だけに現れるとは限りません。消費者が政策を認知し、実際に利用し、その利用が業種指標に反映されるまでには時間がかかる可能性があります。

そのため、政策実施当月の反応だけでなく、1～3か月後の反応も併せて確認しました。

### なぜsynthetic comparatorを追加したのか？

低曝露業種の単純平均一つだけを比較基準にすると、結果が特定の比較群構成に依存する可能性があります。

synthetic comparatorは政策前の動きをより近づけた比較基準です。これにより、主要パターンが比較基準を変えても維持されるかを確認しました。

### なぜplaceboを実施したのか？

全国規模の公開データでは、完全にクリーンな対照群を作ることが困難です。

placeboは、政策がなかった偽時点でも類似したgap変化が現れるかを確認し、主要結果を純粋な政策効果として過剰解釈しないための手段です。

### なぜ融資への示唆まで接続したのか？

政策による売上は、プラットフォーム融資で返済能力の判断を歪める可能性があります。

本プロジェクトでは政策評価自体にとどまらず、政策により増加した売上を、融資限度額・返済条件・モニタリングへどのように反映すべきかまで接続しました。

## 8. Development Notes

本プロジェクトは当初、「消費支援政策後に売上が増加したか」という問いから始まりました。

しかし分析を進める中で、より重要な問いは「増加した売上がどの程度長く残ったか」であることが明確になりました。

政策実施当月の係数はほぼ0に近い値でした。当月反応だけを見ていたなら、政策反応は弱いと結論付けた可能性があります。

しかし、1～3か月lagの係数はすべて正の方向であり、0～3か月の累積反応も正の方向でした。これは、政策反応が当月の急上昇ではなく、数か月にわたり累積する形で現れる可能性を示しています。

次の転換点はsynthetic comparatorとplaceboでした。synthetic comparatorでも方向は維持された一方、placeboでも一部にgap変化が現れました。このため、結果を「純粋な政策効果」として強く主張せず、「政策時期の動学的相対反応」として限定して解釈しました。

最後の転換点は融資への解釈でした。政策による売上は、直近売上に基づく融資限度額を歪める可能性があります。そこで最終メッセージを政策成果評価ではなく、**政策による売上を返済能力として解釈する際には持続性検証が必要である**という方向へ整理しました。

## 9. Limitations

本プロジェクトは、Half-Half系列の消費支援政策による純粋な因果効果を推定したものではありません。

第一に、公開データには加盟店別売上、クーポン使用有無、反復注文、返済結果がありません。そのため、個別加盟店の売上維持率や返済リスクを直接分析できません。

第二に、分析対象は飲食店単独の売上ではなく、宿泊・飲食サービス業を合算した指標です。飲食店プラットフォームの売上と完全に同一の指標ではありません。

第三に、全国規模の政策であるため、完全にクリーンな対照群を作ることが困難です。低曝露業種とsynthetic comparatorは比較基準であり、実験的な対照群ではありません。

第四に、観光需要、季節性、景気回復、年末消費が、政策時期の反応に同時に混在している可能性があります。

第五に、placebo結果は、主要結果を純粋な政策効果として過剰解釈すべきではないという警告を示します。

したがって、本プロジェクトの結果は次のように限定して解釈する必要があります。

> 政策時期に、宿泊・飲食サービス業指標が低曝露業種と比べて1～3か月にわたる正の相対反応を示した。  
> ただし、これを政策の純粋な因果効果や個別加盟店の持続的な返済能力として直接解釈してはならない。

## 12. What This Project Demonstrates

本プロジェクトは、政策による売上増加を単純な売上上昇として解釈せず、持続性・比較基準・融資への示唆まで併せて検討した分析です。

第一に、政策後の売上増加を直ちに持続的な返済能力と解釈せず、政策性需要と持続需要を区別する問題として再定義しました。

第二に、宿泊・飲食サービス業指標と低曝露業種指標の対数gapを構成し、相対反応を分析しました。

第三に、政策変数を単純な0/1ダミーではなく、月別政策強度として定義しました。

第四に、lag modelを使用し、政策実施当月の反応と1～3か月後の反応を分離しました。

第五に、0～3か月の累積反応を中心指標として使用し、政策による反応が当月だけにとどまるのか、数か月残るのかを確認しました。

第六に、synthetic comparator、placebo、robustness検証を通じて、結果をどこまで解釈できるかを確認しました。

最後に、分析結果をプラットフォーム融資における売上ベースの返済能力判断へ接続しました。

本プロジェクトの中心は、単に政策効果を推定したことではなく、**政策によって変動した売上データを、融資意思決定においてどのように慎重に解釈すべきかを示したこと**です。
