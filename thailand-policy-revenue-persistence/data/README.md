# Data

원자료 다운로드 파일은 GitHub에 포함하지 않습니다.

노트북은 `outputs/tables/`에 포함된 파생 패널과 결과표를 사용합니다.

- `final_monthly_model_panel.csv`: 월별 모델 패널
- `policy_calendar_monthly.csv`: 월별 정책 강도
- `dlm_gap_equal.csv`, `dlm_gap_synth.csv`: distributed lag 결과
- `end_centered_event_study_*.csv`: 정책 종료 기준 event study
- `permutation_results.csv`, `robustness_grid_results.csv`: 검증 결과
