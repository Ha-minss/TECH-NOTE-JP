# Data

Raw data files are not tracked in this repository.

Place the Xente CSV files under:

```text
data/raw/
|-- Train.csv
|-- Test.csv
|-- unlinked_masked_final.csv
|-- sample_submission.csv
`-- VariableDefinitions.csv
```

The notebooks also support `XENTE_RAW_DIR` and `XENTE_PROCESSED_DIR` environment variables.
