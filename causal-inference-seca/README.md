# Korea SECA → Northern Kyushu SO₂

韓国の硫黄酸化物排出規制海域（SECA）の導入が、日本の北部九州におけるSO₂濃度へ与えた影響を分析した、環境政策・因果推論プロジェクトです。

本プロジェクトでは、2020年9月に施行された韓国SECA Step 1以降、日本の北部九州にある沿岸観測局のSO₂濃度が、内陸観測局と比べて相対的に低下したかを検証します。分析には、福岡県、佐賀県、長崎県の大気汚染常時監視データを使用し、DID、Event Study、Fixed Effects、placebo test、NO₂を用いた検証、IMO 2020の同時統制、PPML検証などを実施しました。

---

## 1. Overview

本プロジェクトの中心となる問いは、次のとおりです。

> 韓国SECAの導入後、日本の北部九州にある沿岸観測局のSO₂濃度は、内陸観測局と比べて相対的に低下したか？

韓国では、2020年9月から主要港湾周辺において、停泊・係留中の船舶を対象とする低硫黄燃料規制が適用されました。この規制は、船舶燃料の硫黄含有率を引き下げることで、SO₂排出量を削減することを目的としています。

本プロジェクトでは、この政策が韓国内の大気質だけでなく、風や長距離大気輸送を通じて、日本の北部九州の大気質にも影響し得るかを観測データに基づいて検討しました。

主要なDID推定では、SECA導入後、沿岸観測局のSO₂が内陸観測局と比べて約9〜10%低下する方向の結果が確認されました。Event Studyでは、政策実施前から沿岸部と内陸部で明確に異なる事前トレンドが存在したことを示す強い統計的証拠は確認されませんでした。また、placebo testとNO₂を用いた検証も、主要結果と大きく矛盾しませんでした。

ただし、本プロジェクトは政策効果を確定的に証明する分析ではありません。船舶別の実際の運航データや排出量推定値を直接結合しておらず、IMO 2020と韓国SECAが同じ2020年に施行されたという制約もあります。

したがって、結論は次の範囲に限定して解釈します。

> 韓国SECA導入後、北部九州の沿岸地域では、SO₂濃度が内陸地域と比べて相対的に低下するパターンが観測された。

---

## 2. Problem & Objective

SO₂は、船舶燃料に含まれる硫黄成分と直接的に関連する大気汚染物質です。船舶が硫黄含有率の高い燃料を使用するとSO₂排出量が増加し、港湾・沿岸地域の大気質へ影響する可能性があります。

韓国SECAは、主要港湾周辺において船舶燃料の硫黄含有率を引き下げる規制です。本プロジェクトが対象とするStep 1は、2020年9月から停泊・係留中の船舶を対象に適用されました。

しかし、政策実施前後のSO₂平均値を単純に比較するだけでは、SECAの影響を判断することは困難です。SO₂濃度は、次の要因からも影響を受けます。

- 季節性
- 気象条件
- 産業活動とエネルギー使用
- 他の海運規制
- COVID-19流行期における経済・移動活動の変化
- 観測局ごとの固定的な地域特性

そのため、「2020年9月以降にSO₂が低下したか」を確認するだけでは十分ではありません。

本プロジェクトでは、問いを次のように再定義しました。

> 政策実施後、沿岸観測局のSO₂は、内陸観測局と比べて相対的により低下したか？

この比較構造を採用した理由は、海上排出の影響が、内陸よりも沿岸地域で先に、または強く観測される可能性があるためです。内陸観測局は、北部九州全体に共通する時間変化や季節性を比較するための基準群として機能します。

本プロジェクトの目的は、次の3点です。

1. 北部九州の観測局・月パネルデータを構築し、SECA導入前後のSO₂変化を分析する。
2. DIDとEvent Studyを用いて、沿岸部と内陸部の相対的変化が政策実施後にどのように変化したかを推定する。
3. placebo test、NO₂を用いた検証、IMO 2020の同時統制、頑健性検証を通じて、結果が特定のモデル仕様や偶然の政策時点選択だけに依存していないかを確認する。

## 3. Data

分析には、日本の北部九州における大気汚染常時監視データを中心に構成した、観測局・月パネルデータを使用しました。

分析対象地域は、福岡県、佐賀県、長崎県です。分析期間は2017年1月から2021年12月までであり、政策導入時点は2020年9月です。

分析単位は、**観測局 × 月**です。特定の観測局における特定月のSO₂濃度を1つの観測値として扱い、同一観測局を複数月にわたり反復観測するパネル構造です。

主要な分析基準は、以下のとおりです。

| 項目 | 内容 |
|---|---|
| 分析地域 | 福岡県、佐賀県、長崎県 |
| 分析期間 | 2017年1月〜2021年12月 |
| 分析単位 | 観測局・月パネル |
| 観測局数 | 70局 |
| 月数 | 60か月 |
| 主要分析の観測値数 | 3,829件 |
| 政策時点 | 2020年9月 韓国SECA Step 1 |
| 主な結果変数 | 月平均SO₂濃度 |
| 主な比較構造 | 沿岸観測局 vs 内陸観測局 |

リポジトリ内の`data/df_iv.csv`は、GitHub上で再現可能な分析を行うために必要なカラムを整理した共有用パネルデータです。

このファイルには、主要分析期間より広い範囲の観測値や補助変数が含まれる場合があります。分析スクリプトでは、Step 1の分析期間と有効観測条件に基づき、必要な分析標本を再構築します。

データは、大きく5つの役割に分かれます。

### 3.1 大気汚染データ

SO₂とNO₂の観測値を含みます。

SO₂は、韓国SECAの直接的な規制メカニズムと結びつく主要な結果変数です。NO₂はSECAが直接対象とする汚染物質ではないため、validation checkの補助的な結果変数として使用しました。

### 3.2 観測局位置情報

沿岸観測局と内陸観測局を分類するために使用しました。

単純な行政区域区分ではなく、以下の情報を活用しました。

- 海岸線までの距離
- 釜山港までの距離
- 海岸方向への開放性

### 3.3 気象データ

SO₂濃度は大気拡散条件の影響を受けるため、以下の変数を統制しました。

- 気温と露点温度の差
- 気圧
- 混合層高度

### 3.4 経済活動・エネルギー関連データ

以下の変数を使用しました。

- ディーゼル価格
- 鉱工業生産指数
- 火力発電関連指標

これらの変数はSO₂濃度と同じ月に同時決定される可能性があるため、主要分析では1か月lagを適用しました。

### 3.5 HYSPLIT Back Trajectory補助分析

この分析は、九州の観測局へ到達した空気塊が、過去72時間にどのような経路を通過したかを確認するためのものです。

DIDの主要な識別変数ではありませんが、韓国SECA周辺を通過する大気輸送経路が実際に存在し得ることを示す補助的な根拠として使用しました。

主要な結果変数は、月平均SO₂濃度です。

一部の観測値には0が含まれるため、基本分析では`log(1 + SO₂)`を使用しました。ただし、ログ変換の結果だけに依存しないように、SO₂の原水準値を使用したOLSとPPMLによる検証も追加しました。

## 4. Method / System Design

主要な分析方法は、Difference-in-Differences（DID）です。

政策実施前後の平均値を単純に比較するだけでは、SECA効果と他の時間変化を区別することが困難です。

そこで本プロジェクトでは、沿岸観測局を処置群、内陸観測局を比較群とし、政策実施前後における相対的な変化を比較しました。

基本モデルは、以下の要素で構成されます。

- 沿岸観測局ダミー
- 政策実施後ダミー
- 沿岸観測局 × 政策実施後の交差項
- 観測局Fixed Effects
- 年月Fixed Effects
- 気象・経済活動の統制変数
- 観測局単位でclusterした標準誤差

観測局Fixed Effectsは、各観測局がもともと持つ固定的な差異を統制します。

たとえば、次の要素は観測局ごとに異なる可能性があります。

- 地理的位置
- 周辺環境
- 長期的な地域排出構造

年月Fixed Effectsは、特定月にすべての観測局へ共通して影響するショックを統制します。

これには、以下が含まれます。

- 季節性
- 広域的な景気変動
- 全国的な規制変更
- 特定月に共通する気象パターン

関心のある係数は、政策実施後に沿岸観測局のSO₂が内陸観測局と比べて追加的にどの程度変化したかを示します。

この係数が負であれば、SECA導入後に沿岸地域のSO₂が内陸地域と比べて相対的に低下したことを意味します。

DIDの主要な仮定は、政策実施前に沿岸部と内陸部のトレンドが大きく異ならないことです。

この仮定を確認するため、Event Studyを実施しました。政策導入時点である2020年9月を基準に、前後12か月の動学的変化を確認し、政策導入直前1か月を基準時点としました。

追加検証は、以下の方向で設計しました。

1. 政策実施前に偽の政策時点を設定しても効果が確認されるか。
2. SO₂ではなくNO₂を結果変数としても同じ効果が確認されるか。
3. IMO 2020と韓国SECAの効果が時期的に混在していないか。
4. 気象・経済活動の統制変数構成を変更しても結果が維持されるか。
5. 沿岸観測局の定義を変更しても結果の方向が維持されるか。
6. ログ変換ではなくSO₂原水準値を使用しても結果の方向が維持されるか。
7. 一部観測局・期間における0値の集中が結果を左右していないか。

この設計の中心は、単一の回帰結果を提示することではありません。

政策効果に見えるパターンが、複数の代替説明や検証を通過するかを確認することです。

## 5. Implementation

本プロジェクトでは、論文形式の分析をGitHub上で再現可能な形に整理するため、分析フローを実行スクリプトと共通モジュールへ分離しました。

実装フローは、以下のとおりです。

1. 共有用パネルデータを読み込む。
2. 分析期間を2017年1月から2021年12月までに限定する。
3. SO₂の有効観測率と必要な統制変数に基づいて分析標本を構成する。
4. SO₂をログ変換し、政策実施後ダミーと沿岸観測局ダミーを組み合わせてDID変数を作成する。
5. 基本DID分析を実行する。
6. Event Studyを実行し、政策実施前後の動学的変化を確認する。
7. validation checkと頑健性検証を実行する。
8. 結果表とグラフを`outputs/`へ保存する。

主要な実行スクリプトは、以下のとおりです。

### `scripts/01_main_did_step1.py`

基本DID分析を実行します。

以下の仕様を比較します。

- Fixed Effectsのみを含む仕様
- 気象統制を追加した仕様
- 気象統制と経済活動lagを含む仕様

### `scripts/02_event_study_step1.py`

2020年9月を基準としてEvent Studyを実行し、政策実施前後の係数とグラフを生成します。

### `scripts/03_robustness_suite_step1.py`

以下の主要な頑健性検証を実行します。

- placebo test
- 統制変数の変更
- 経済活動lagの変更
- 標準誤差推定に関する検証

### `scripts/04_level_so2_ols_ppml.py`

ログ変換結果だけに依存しないよう、SO₂原水準値を用いたOLSとPPML検証を実施します。

### `scripts/05_treatment_defs_step1.py`

沿岸観測局の定義を変更した場合にも、結果が維持されるかを確認します。

### `scripts/00_prepare_df_iv_for_github.py`

共有可能な分析用パネルデータを再構築するための補助スクリプトです。

共通コードである`step1_did/`モジュールには、以下が含まれます。

- データ読み込み
- 前処理
- DID推定
- ユーティリティ関数
- wild cluster bootstrap補助関数

この構造により、すべての分析を1つのnotebookへまとめるのではなく、各分析段階を独立して実行・再現できるようにしました。

## 6. Evaluation

評価は、単に「係数が統計的に有意か」だけを見るようには設計していません。

環境政策分析では、次の要素を同時に確認する必要があります。

- 推定結果の方向
- 事前トレンド
- placebo test
- 代替結果変数
- 他政策との時期的重複
- データ品質上の問題

基本DID分析では、3つの主要仕様すべてにおいて、沿岸地域のSO₂が内陸地域と比べて相対的に低下する方向の結果が確認されました。

| モデル仕様 | 推定結果 |
|---|---:|
| Fixed Effectsのみ | 約-0.098 log point |
| 気象統制を追加 | 約-0.100 log point |
| 気象統制と経済活動lagを追加 | 約-0.101 log point |

ログ結果変数を基準にすると、これはおおむね9〜10%程度の相対的低下に相当します。

Event Studyでは、政策導入前の沿岸部と内陸部の事前トレンドが統計的に大きく異なっていたことを示す証拠は確認されませんでした。

| 検定 | p-value |
|---|---:|
| 政策実施前の全期間に対する結合検定 | 0.711 |
| 政策実施直前の近接期間に対する検定 | 0.734 |

Validation checkでは、以下を確認しました。

### 6.1 Placebo Test

政策導入前の複数時点を偽の政策導入時点として設定したplacebo testでは、統計的に有意な効果は確認されませんでした。

### 6.2 NO₂検証

SO₂ではなくNO₂を結果変数として使用した検証では、推定効果は0に近く、統計的に有意ではありませんでした。

これは、主要結果が単純な大気質全般の改善ではなく、SO₂とより直接的に関連するパターンである可能性を補助的に示します。

### 6.3 IMO 2020の同時統制

2020年1月に施行されたIMO 2020と、2020年9月に施行された韓国SECAを同一モデル内で考慮しました。

この検証では、韓国SECAに対応する係数は負の方向を維持した一方、IMO 2020に対応する係数は統計的に明確ではありませんでした。

### 6.4 その他の頑健性検証

以下の検証を実施しました。

- 統制変数の組み合わせ変更
- 経済活動lagの変更
- 標準誤差推定方法の変更
- 沿岸観測局定義の変更
- 一部の0値集中観測値の除外
- SO₂原水準値を用いたOLS
- PPML推定

主要な検証では、推定係数の符号はおおむね負の方向を維持しました。

したがって、本プロジェクトの結果は次のように要約できます。

> 韓国SECA Step 1導入後、北部九州の沿岸観測局におけるSO₂濃度は、内陸観測局と比べて相対的に低下する方向を示した。  
> ただし、すべての仕様で強い統計的有意性が維持されたわけではないため、結果は限定的に解釈する必要がある。

## 7. Key Design Decisions

### 7.1 なぜDIDを使用したのか？

政策実施前後の平均値だけを比較すると、SECA効果と以下の要因を区別できません。

- 季節性
- 気象変化
- 経済活動の変化
- 他の海運規制

そこで、沿岸観測局と内陸観測局の相対的変化を比較するDID構造を使用しました。

この選択は、単純な平均差ではなく、政策実施後に沿岸地域で内陸地域と比べて追加的な変化が生じたかを確認するためのものです。

### 7.2 なぜ沿岸観測局を処置群としたのか？

SECAは船舶燃料規制であるため、海上排出の影響を受ける可能性がより高い地域は沿岸観測局です。

内陸観測局は、同じ北部九州の中で季節性や共通ショックを比較するための基準群として機能します。

ただし、沿岸性を単純な行政区域だけで定義することは困難です。

本プロジェクトでは、以下を併せて考慮しました。

- 海岸線までの距離
- 釜山港までの距離
- 海岸方向への開放性

その後、代替的な沿岸定義を用いた頑健性検証も実施しました。

### 7.3 なぜ経済活動変数へlagを適用したのか？

経済活動とSO₂は、同じ月に同時に変化する可能性があります。

同月の経済活動変数をそのままモデルへ含めると、同時性の問題が生じる可能性があります。

そのため、以下の変数には1か月lagを適用しました。

- ディーゼル価格
- 鉱工業生産指数
- 火力発電関連指標

これは、同月のショックを直接統制変数へ含めるよりも、保守的な選択です。

### 7.4 なぜNO₂検証を追加したのか？

韓国SECAは燃料中の硫黄含有率を規制する政策であるため、SO₂とより直接的に関連します。

NO₂でも同じ方向の強い効果が確認された場合、主要結果がSECAによるものではなく、一般的な大気質改善による可能性があります。

NO₂検証で明確な効果が確認されなかったことは、SO₂結果の解釈を補助するvalidation checkとして使用しました。

### 7.5 なぜIMO 2020を同時に考慮したのか？

IMO 2020は2020年1月、韓国SECA Step 1は2020年9月に施行されました。

両規制の実施時期が近いため、韓国SECAの結果を解釈する際にはIMO 2020を考慮する必要があります。

本プロジェクトでは、同一モデル内で以下を同時に推定しました。

- IMO 2020以降における沿岸地域の変化
- 韓国SECA以降における沿岸地域の変化

### 7.6 なぜPPML検証を追加したのか？

主要分析では`log(1 + SO₂)`を使用します。

しかし、0値を含むデータでは、ログ変換の選択によって結果の解釈が変わる可能性があります。

そのため、SO₂原水準値を用いたOLSとPPML検証を追加し、主要結果がログ変換方法だけに依存していないかを確認しました。

## 8. Development Notes

本プロジェクトは、当初から現在の分析構造で始まったわけではありません。

初期の問いは、次のようなものでした。

> 韓国SECA導入後、九州のSO₂は低下したか？

しかし、SO₂の月別変化を確認する中で、単純な政策前後比較は危険であることが明確になりました。

SO₂には明確な季節性があり、沿岸部・内陸部の双方で、春から初夏にかけて高く、秋から冬にかけて低くなるパターンが確認されました。

そこで、分析の問いを変更しました。

```text
政策実施後にSO₂は低下したか？
```

から、

```text
政策実施後、沿岸観測局のSO₂は内陸観測局と比べて相対的に低下したか？
```

へ転換しました。

この転換が、本プロジェクトの中心です。

次に、DIDの前提条件を確認するため、Event Studyを追加しました。

政策実施前から沿岸部と内陸部のトレンドが大きく異なっていた場合、DID結果を政策効果として解釈することが困難になるためです。

分析の途中では、2020年に韓国SECAだけでなくIMO 2020も存在するという問題がありました。

そのため、2つの政策時点が推定結果へ混在する可能性を抑えるため、IMO 2020と韓国SECAを同一モデル内で確認しました。

また、一部の内陸観測局と特定期間でSO₂の0値が集中する問題も確認されました。

これをそのまま使用すると、特定の観測局や期間が推定結果を大きく左右する可能性があります。そのため、該当する観測局または期間を除外した保守的な再推定も実施しました。

最終的に、本プロジェクトは単一のDID結果を示す分析から、事前トレンドと複数の代替説明を併せて確認する政策評価プロジェクトへ発展しました。

## 9. Limitations

本プロジェクトは、韓国SECAが北部九州のSO₂へ影響した可能性を、観測データに基づいて検討した分析です。

ただし、以下の限界があります。

1. **船舶別のAIS運航データや排出量推定データを直接結合していません。**  
   そのため、実際の排出削減量と九州観測局のSO₂変化を直接結びつけた分析ではありません。

2. **HYSPLIT back trajectoryは補助分析です。**  
   高解像度の大気化学モデルを用いた直接的なメカニズム検証ではありません。

3. **IMO 2020と韓国SECA Step 1はいずれも2020年に施行されています。**  
   同一モデルで両時点を考慮しましたが、両規制の効果を完全に分離することには限界があります。

4. **一部の観測局と期間でSO₂の0値が集中していました。**  
   保守的な再推定でも符号はおおむね維持されましたが、低濃度区間における測定・報告単位の問題は推定精度を制限する可能性があります。

5. **分析地域は福岡県、佐賀県、長崎県に限定されています。**  
   結果を日本全体または東アジア全体へ一般化することは困難です。

6. **沿岸観測局の定義には不確実性があります。**  
   海岸線までの距離、釜山港までの距離、海岸方向への開放性を使用しましたが、実際の大気輸送や曝露強度を完全に反映するとは限りません。

今後の改善方向は、以下を結合することです。

- AISに基づく船舶運航データ
- 船舶排出量推定データ
- より高い時間解像度を持つ大気質データ
- 標準化された品質管理手順

## 12. What This Project Demonstrates

本プロジェクトは、単にDIDを実行した分析ではなく、観測データを用いて政策効果を検討する際に必要な分析設計を示しています。

1. **現実の政策課題を分析可能な比較構造へ変換しました。**  
   「政策実施後にSO₂が低下したか」という問いを、「政策実施後、沿岸観測局のSO₂が内陸観測局と比べて相対的に低下したか」というDIDの問いへ再定義しました。

2. **観測局・月パネルデータを構築しました。**  
   大気汚染データ、気象データ、経済活動データ、観測局位置情報を結合し、分析可能なデータ構造を構築しました。

3. **DIDとEvent Studyを用いて政策実施前後の相対的変化を推定し、事前トレンドを確認しました。**

4. **複数の検証により、結果が単一のモデル仕様だけに依存していないかを確認しました。**  
   placebo test、NO₂検証、IMO 2020の同時統制、頑健性検証、PPML検証を実施しました。

5. **結果を過度に一般化せず、限界と併せて解釈しました。**  
   主要結果は約9〜10%の相対的低下方向を示しましたが、これを確定的な因果効果とは断定せず、観測データから確認された限定的なシグナルとして解釈しました。

本プロジェクトが示す中心的な能力は、政策上の問いをデータ分析問題へ構造化し、比較群と識別仮定を設定し、複数の方法で結果を検証し、解釈可能な範囲内で結論を提示する能力です。
